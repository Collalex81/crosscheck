Cross-check two spreadsheets

Reconcile any two files (CSV or XLSX) — match, ambiguous, no-match and only-in-anchor — with configurable status rules and a formatted Excel export. Nothing leaves your device.

🔒 100% offline 📄 Single file 📊 Excel export
1Load both files — the anchor (fixed structure) and the file to cross-reference. CSV or XLSX, any layout.
2Map keys — for each priority pick the column that identifies a row. Rename a priority by editing its label.
3Pick columns — check the columns you want in the report on each side, then drag the chips to set their order.
4Cross-check & export — review the results, filter, then export a formatted Excel.

How to read the match: each client row was cross-checked against FC by the first available key in order Serial → EO → Part Number → SKU / Code. When a row matches several FC records (ambiguous), each match appears on its own row, labeled 1 of N. FC only = FC records the client didn't report. Choose above which FC and client columns to show; the report and the exported Excel use that selection.
